### Script pre-commit
Este script rejeita commits de arquivos com nome `ft_*.c`, `ft_*.h` e `libft.h` enquanto estiverem fora da norma.

#### Instalação:
```sh
cp scripts/pre-commit .git/hooks/
chmod +x .git/hooks/pre-commit
```
